§ 01 / Co vyvíjíme

Čtyři disciplíny, jeden tým.

Celý stack
in-house
Disciplína / 01

Frontend vývoj

Moderní, rychlé a responzivní uživatelské rozhraní. Accessibility WCAG 2.2 AA jako standard.

  • Vue.js 3 / Nuxt.js 3
  • React 19 / Next.js
  • TypeScript 5
  • Headless architektura
  • Tailwind, CSS modules
Disciplína / 02

Backend vývoj

Robustní a škálovatelné serverové aplikace. Testy, monitoring, CI/CD od prvního commitu.

  • Node.js / Express / Nest
  • PHP / Laravel / Symfony
  • Python / Django / FastAPI
  • REST & GraphQL API
  • PostgreSQL, Redis, Elastic
Disciplína / 03

Integrace

Propojení systémů a automatizace procesů. ERP, CRM, platební brány, logistika.

  • ERP (SAP, Abra, Money, Pohoda)
  • CRM (HubSpot, Salesforce)
  • Platební brány (Stripe, GP)
  • Dopravci (Zásilkovna, PPL, DPD)
  • Marketplaces (Heureka, Zbozi)
Disciplína / 04

Mobilní aplikace

Cross-platform aplikace pro iOS a Android. Jedna codebase, dvě storu.

  • React Native
  • Expo + OTA updates
  • PWA aplikace
  • Hybridní řešení (Capacitor)
  • App Store / Google Play deploy
